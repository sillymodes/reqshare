# ReqShare

Show your API, don't just describe it. Create runnable API example pages and share them with a link.

## Overview

ReqShare lets developers create beautiful, shareable, runnable API request collections and share them via a single URL. Think "CodePen for API calls."

## Tech Stack

- **Frontend:** Vanilla HTML/CSS/JS on GitHub Pages
- **Backend:** Cloudflare Workers + Workers KV
- **CI/CD:** GitHub Actions
